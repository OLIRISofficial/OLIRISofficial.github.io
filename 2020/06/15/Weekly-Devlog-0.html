      Weekly Devlog #0

      Jun 15, 2020

      Welcome Back To OLIRIS . Today I Am Going To Talk About What I’ve Been Doing For The Last Couple Of Weeks And Show The Games’ Current Development State. So Let’s Begin.

      Tales Of War

      First Up I Almost Finished Placement System Of The Game. I Set Up A Grid System With Godot’s In Engine 3D Tiles GridMap. Then I Added Boundary System That Changes Building’s Material Depending On If It’s Placeable Or Not.

      I Re Factored The Selection System To Be More Efficient And Easy To Read.

      With The Re Factoring Process I Also Fixed A Bug In The UI. The Bug Was If You Clicked Somewhere Empty On The Panels It Would’ve De Select Everything.

      I Also Added Health System With Health Bars On Top Of Units And Buildings. It Still Lacks Design Tho.

      Tales Of Heroes

      Now I Started This Project 2 Weeks Ago So The Game Is In A Very Rough State. Please Keep That In Mind.

      Switching Between First Person To Third Person (Please Don’t Mind How The Cross-air Looks.)

      I Added Ability System Which Makes The Player Do Some Stuff Like Double Jumping, Sliding Left Or Right, Dashing Etc. You Can Active Them With Keyboard Or Pressing The Abilities Button.

      I Added Health System With A Health Bar. It Also Lacks Design Too.

      And Actually That Was All The Things That I Did For Tales Of Heroes This Week. Hopefully Next Week Will Be More Productive Than This Week.
